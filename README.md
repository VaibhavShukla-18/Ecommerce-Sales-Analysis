# 📊 E-Commerce Sales Analysis Using Python

## 📌 Project Overview

The E-Commerce Sales Analysis project is a beginner-friendly data analytics project developed using Python. It demonstrates how to load, clean, analyze, and visualize sales data using the Pandas and Matplotlib libraries.

The project helps identify business trends, product performance, regional sales distribution, and monthly revenue through interactive data analysis and visualizations.

---

## 🎯 Objectives

- Load and analyze sales data from a CSV file
- Clean and preprocess the dataset
- Calculate important business metrics
- Visualize sales using charts
- Generate meaningful business insights

---

## 🛠️ Technologies Used

- Python 3.12
- Pandas
- NumPy
- Matplotlib
- Visual Studio Code

## 📊 Visualizations

### 1. Product-wise Sales
Displays the total sales for each product using a bar chart.

### 2. Monthly Sales Trend
Shows monthly sales growth using a line chart.

### 3. Regional Sales Distribution
Illustrates sales contribution from each region using a pie chart.

---

## 📌 Business Insights

- Laptops generated the highest sales revenue.
- The North region contributed the largest share of total sales.
- March recorded the highest monthly sales.
- April recorded the lowest monthly sales.
- The dataset contains 100 sales records across multiple products and regions.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Ecommerce-Sales-Analysis.git
```

Move into the project directory

```bash
cd Ecommerce-Sales-Analysis
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python main.py
```
