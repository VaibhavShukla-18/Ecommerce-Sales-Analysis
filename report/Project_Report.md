# E-Commerce Sales Analysis Using Python

## 1. Introduction

The E-Commerce Sales Analysis project is designed to analyze sales data using Python. The project demonstrates the use of data analysis and visualization techniques to extract meaningful business insights.

---

## 2. Objectives

- Load sales data
- Clean the dataset
- Analyze sales performance
- Visualize business trends
- Generate insights

---

## 3. Dataset Description

The dataset contains 100 sales records with the following columns:

- Date
- Product
- Quantity
- Price
- Customer_ID
- Region
- Total_Sales

---

## 4. Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Visual Studio Code

---

## 5. Methodology

### Step 1
Load dataset using Pandas.

### Step 2
Check missing values.

### Step 3
Remove duplicate records.

### Step 4
Perform exploratory data analysis.

### Step 5
Create charts.

### Step 6
Generate business insights.

---

## 6. Charts

### Bar Chart
Shows product-wise sales.

### Line Chart
Shows monthly sales trend.

### Pie Chart
Shows region-wise sales distribution.

---

## 7. Business Insights

- Laptop generated the highest sales.
- North region generated the maximum revenue.
- March achieved the highest sales.
- April recorded the lowest sales.

---

## 8. Conclusion

Python provides an efficient platform for analyzing sales data. Using Pandas and Matplotlib, businesses can easily visualize trends and make informed decisions.
